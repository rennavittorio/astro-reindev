# reindev.it

Personal website and portfolio of Vittorio Renna — fullstack developer with a neuroscience background.

Live: [reindev.it](https://reindev.it)

---

## Stack

- [Astro 5](https://astro.build) — static-first framework with React islands
- [React 19](https://react.dev) — used selectively for interactive components
- [Tailwind CSS v4](https://tailwindcss.com) — utility-first styling via Vite plugin
- TypeScript throughout

## Notable bits

- AsciiTrail — custom canvas animation that reacts to mouse movement; falls back to an autonomous wandering cursor on touch devices
- Island architecture: only interactive components ship JavaScript, the rest is zero-JS HTML
- Deployed on [Netlify](https://www.netlify.com/)
